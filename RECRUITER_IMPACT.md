## 6. Conversion Funnel Estimate

Assuming 100 cold recruiter profile views per month before the update:

- **Before:** 100 profile views → 10 clicks to portfolio/resume → 2 connection requests → 1 screen call
- **After:** 100 profile views → 25 clicks to portfolio/resume → 8 connection requests → 3–4 screen calls

That is roughly a **3x improvement in screen-call conversion** from the same number of profile views.
