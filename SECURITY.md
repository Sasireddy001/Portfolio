# Security Policy

## Supported Versions

Only the latest `main` branch of this portfolio repository is actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability in this portfolio or any linked project, please report it responsibly:

1. **Do not open a public issue.** Public issues can expose vulnerabilities before a fix is available.
2. Email the details to **sasidharmopuru@gmail.com** with the subject line `[Security] Portfolio`.
3. Include a clear description, steps to reproduce, and any potential impact.

I will acknowledge the report within 48 hours and aim to provide a resolution or timeline within 7 days.

## Security Practices

- No API keys, passwords, or personal access tokens are committed to this repository.
- External links (LinkedIn, GitHub, credential verification) use `rel="noopener"` and open in new tabs.
- The contact email is used for professional inquiries only.
